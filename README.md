# mythos2
A Model-View-ViewModel (MVVM) library for Android apps. 

## Setup
```groovy
repositories {
    maven { url "https://jitpack.io" }
}
dependencies {
    implementation "com.github.jhavatar:mythos2:0.1.5"
}
```
